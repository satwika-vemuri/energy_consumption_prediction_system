Compilation/Execution Instructions:
1. Run rpi_pub_and_sub.py on RPi 
2. Run vm_pub_and_sub.py on local machine or VM. 
   This should start a Flask app on your machine on a specific port (ex: localhost:5000) 
3. Visit the Flask app on your local browser, enter a city name, and click the "Retrieve Weather!" button.
4. Results should display on the Flask app.
